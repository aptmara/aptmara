## 今やっていること

- `Unity のゲーム試作`  
  小さく動くところまで一気に作って、ゲームループの気持ちよさを先に検証します。
- `デスクトップ向けツール`  
  日常作業を少し速くする Windows ツールを C# で作ります。
- `Web ツールとブラウザ拡張`  
  ブラウザ拡張や小規模 Web アプリで、運用や閲覧の摩擦を減らす方向も触っています。
